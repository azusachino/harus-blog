## things

- 这个月开销不小, 使用 mac mini 运行容器的体验还是太差了, 于是, 最终决定再买一台迷你主机, 专门用来跑容器, 也是买了 64G/1T 的高配版本, 体验还不错, 只是耗电确实比 macmini 高了不少; 以及, 不知道何缘由, 决定购入了电视, 虽然已经有两个用不上的显示器了; 总体体验也还不错, 小米的 s mini 2025 55 英寸, 比想象中大很多, 搭配 PS5 很优秀.
- 算是把 cloudflare R2 利用上了, 实现了一个备份本地文件的小工具, 每天定时跑一下, 根本用不完免费的 RW 额度, 不用担心 _HOMELAB_ 意外当机了.
- 从 podman 换回 docker 了, 其实一直没花心思去学习 rootless 相关的概念, 以及 podman 在不同 distro 上版本不太一致, 使用起来问题蛮多, 包括很多程序都是基于 /var/run/docker.sock 去实现的, 以及权限上也很多奇奇怪怪的报错, 换回 docker, 虽然命令都需要 sudo, 但没有遇到任何奇怪的问题了.
- 参考偏实战的专栏重新学习 Linux 的运维排查错误相关问题, 虽然按照章节一点点往下推进了, 但是缺乏针对不同模块的最终检测, 很难说掌握了多少知识点, 看过亦是忘过…
- 3 月的天气确实无法预测, 有雨有雪, 也有接近 30 度左右的高温. 下雨天就在家蜗居了, 天气合适的时候坚持出门跑了几天, 晨跑的效果很不错, 也很适合我, 空腹跑步最棒
- 周末依旧是寿喜烧, 乱炖面为主食, 而且中午总是会吃的很撑, 下午又不会出门, 就导致一整天胃都不舒服, 这块是接下来需要调整的重点区域
- 本来总是想使用 dotenv 以及 cron 三方库来实现定时任务, 后面发现了 systemd-timer 这个神器, 虽然之前也看到很多事例, 但一直没有提起注意. 经过 copilot 的指点, 现在也能善用 systemd 了, 执行 oneshot, simple, timer 都很方便快捷, 而且也不用担心机器重启.
- 尝试搭建了 k3s cluster, 才发现需要稳定的公网 IP 才可以, tailscale 的这种 overlay 组网, 说实话还不太清楚具体的逻辑依据是什么, 后续也需要专案调查一下; 目前 docker + portainer 够用了, caddy 反向代理也很优秀, 不需要额外处理 SSL 的问题了.
